Basic Image Processing: Hands-On Fundamentals

Now let's get hands-on with the practical skills! We'll start from the very beginning and build up systematically. For complete working code, see the code examples 02-10.

Loading, Displaying, and Saving Images

The foundation of all image processing:

import cv2

# Load an image
img = cv2.imread('photo.jpg')

# Always check if image loaded
if img is None:
    print("Error: Could not load image")
    exit()

print(f"Shape: {img.shape}")  # (height, width, channels)

# Display image
cv2.imshow('Original Image', img)
cv2.waitKey(0)
cv2.destroyAllWindows()

# Save a copy
cv2.imwrite('output.jpg', img)

OpenCV Image I/O Documentation

Color Space Conversions

Working with different color representations:

# BGR ‚Üí Grayscale
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# BGR ‚Üí RGB (for matplotlib)
rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)

# BGR ‚Üí HSV (better for color-based tasks)
hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)

Remember: OpenCV uses BGR by default, not RGB!

Image Transformations

Resizing
# Resize to specific dimensions
resized = cv2.resize(img, (800, 600))

# Resize by scale factor
half = cv2.resize(img, None, fx=0.5, fy=0.5)

# Use appropriate interpolation:
# INTER_AREA for shrinking
# INTER_CUBIC for enlarging
enlarged = cv2.resize(img, (1600, 1200), 
                     interpolation=cv2.INTER_CUBIC)
Rotation and Flipping
# Rotate 90 degrees
rotated = cv2.rotate(img, cv2.ROTATE_90_CLOCKWISE)

# Flip horizontally
flipped = cv2.flip(img, 1)

# For arbitrary angles:
center = (img.shape[1]//2, img.shape[0]//2)
M = cv2.getRotationMatrix2D(center, 45, 1.0)
rotated_45 = cv2.warpAffine(img, M, (img.shape[1], img.shape[0]))
Cropping
# Crop using NumPy slicing: img[y1:y2, x1:x2]
cropped = img[100:400, 200:500]

# Crop center region
h, w = img.shape[:2]
center_crop = img[h//4:3*h//4, w//4:3*w//4]

Basic Filters and Blurring

Blur images to reduce noise or create artistic effects:

# Gaussian Blur (most common)
gaussian = cv2.GaussianBlur(img, (5, 5), 0)

# Median Blur (removes salt-and-pepper noise)
median = cv2.medianBlur(img, 5)

# Bilateral Filter (preserves edges)
bilateral = cv2.bilateralFilter(img, 9, 75, 75)

Quick guide:

  • Gaussian: General purpose, natural-looking
  • Median: Best for impulse noise
  • Bilateral: Smooths while keeping edges sharp

Brightness and Contrast

# Brighten image
brighter = cv2.add(img, 50)

# Adjust both brightness and contrast
# Formula: new = alpha * img + beta
adjusted = cv2.convertScaleAbs(img, alpha=1.3, beta=50)

Thresholding

Convert to binary (black and white):

gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# Simple binary threshold
ret, binary = cv2.threshold(gray, 127, 255, cv2.THRESH_BINARY)

# Otsu's automatic threshold
ret, otsu = cv2.threshold(gray, 0, 255, 
                          cv2.THRESH_BINARY + cv2.THRESH_OTSU)

# Adaptive threshold (varying lighting)
adaptive = cv2.adaptiveThreshold(gray, 255,
                                cv2.ADAPTIVE_THRESH_GAUSSIAN_C,
                                cv2.THRESH_BINARY, 11, 2)

Pixel Manipulation

# Access single pixel [y, x]
pixel = img[100, 200]  # Returns [B, G, R]

# Modify pixel
img[100, 200] = [255, 0, 0]  # Set to blue

# Modify region
img[0:100, 0:100] = [0, 255, 0]  # Green square

# Remove channel
img[:, :, 2] = 0  # Remove all red

Image Compositing

Combine multiple images:

# Alpha blending (70% img1 + 30% img2)
blended = cv2.addWeighted(img1, 0.7, img2, 0.3, 0)

# Picture-in-picture overlay
background[y:y+h, x:x+w] = foreground

Advanced Image Processing: Filters & Convolution

Understanding Convolution

Convolution is the mathematical operation behind most image filters. Think of it as a sliding window that examines each pixel with its neighbors.

How it works:

  1. Place a kernel (small matrix) over a pixel
  2. Multiply kernel values √ó pixel values
  3. Sum all the products
  4. Replace center pixel with the sum
  5. Move to next pixel and repeat
Convolution in OpenCV
import numpy as np

# Define a custom kernel
kernel = np.array([[1, 2, 1],
                   [2, 4, 2],
                   [1, 2, 1]], dtype=np.float32)

# Normalize (sum = 1)
kernel = kernel / np.sum(kernel)

# Apply convolution
filtered = cv2.filter2D(img, -1, kernel)

cv2.filter2D() Documentation

Custom Kernel Examples

Edge Enhancement:

edge_kernel = np.array([[-1, -1, -1],
                        [-1,  8, -1],
                        [-1, -1, -1]])

Sharpen:

sharpen_kernel = np.array([[ 0, -1,  0],
                           [-1,  5, -1],
                           [ 0, -1,  0]])

Emboss:

emboss_kernel = np.array([[-2, -1,  0],
                          [-1,  1,  1],
                          [ 0,  1,  2]])

Advanced Blur Techniques

Now that we understand convolution, we can see that Gaussian blur is just convolution with a Gaussian kernel!

# Compare all blur methods
gaussian = cv2.GaussianBlur(img, (9, 9), 0)
median = cv2.medianBlur(img, 9)
bilateral = cv2.bilateralFilter(img, 9, 75, 75)
box = cv2.blur(img, (9, 9))

Sharpening Filters

Enhance edges and details:

# Simple sharpen
sharpen = np.array([[0, -1, 0],
                    [-1, 5, -1],
                    [0, -1, 0]])
sharpened = cv2.filter2D(img, -1, sharpen)

# Unsharp mask (more control)
def unsharp_mask(image, amount=1.5):
    blurred = cv2.GaussianBlur(image, (5, 5), 1.0)
    sharpened = cv2.addWeighted(image, 1.0 + amount, 
                                blurred, -amount, 0)
    return sharpened

sharp = unsharp_mask(img, amount=1.5)

Warning: Don't over-sharpen - it amplifies noise!

Edge Detection

Edge detection identifies boundaries where intensity changes rapidly.

Sobel Operator
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# Horizontal and vertical edges
sobel_x = cv2.Sobel(gray, cv2.CV_64F, 1, 0, ksize=3)
sobel_y = cv2.Sobel(gray, cv2.CV_64F, 0, 1, ksize=3)

# Combine both directions
sobel = cv2.magnitude(sobel_x, sobel_y)

cv2.Sobel() Documentation

Laplacian
# Detects edges in all directions
laplacian = cv2.Laplacian(gray, cv2.CV_64F)
Canny Edge Detection ‚≠ê

Most sophisticated method with best results:

# Always blur first!
blurred = cv2.GaussianBlur(gray, (5, 5), 0)

# Canny edge detection
edges = cv2.Canny(blurred, 50, 150)

cv2.imshow('Canny Edges', edges)

cv2.Canny() Documentation

Quick comparison:

  • Sobel: Fast, directional, can be noisy
  • Laplacian: All directions, very sensitive to noise
  • Canny: Best quality, most commonly used ‚≠ê

üí° Pro Tip: Always apply Gaussian blur before edge detection to reduce noise! This dramatically improves results.
